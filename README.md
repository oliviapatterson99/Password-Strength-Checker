## ğŸ“– Overview

Password Strength Checker is a Ruby on Rails project that mimics the basic behavior of a password manager. Users can store passwords in a database and see whether each password meets basic strength requirements.

This project was built as a personal challenge and as a portfolio piece to demonstrate foundational Rails skills, MVC structure, and CRUD functionality.
